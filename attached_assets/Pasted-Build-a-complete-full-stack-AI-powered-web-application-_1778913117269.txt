Build a complete full-stack AI-powered web application named “AI Career Guidance System” using React.js for frontend, Node.js + Express.js for backend, MongoDB for database, and Python ML models for predictions and recommendations.

The website should be modern, responsive, professional, and portfolio-level with smooth animations, dashboards, charts, cards, gradients, and mobile-friendly design.

Main Objective:
The system should allow students to:

* Create an account and login
* Upload resumes (PDF/DOCX)
* Enter academic and skill details
* Analyze resume content using AI
* Predict performance and placement eligibility
* Recommend best career domains
* Group students by skill level
* Generate personalized AI career guidance and improvement suggestions

Machine Learning Models:

1. Linear Regression:
   Predict student performance score based on:

* Attendance
* Coding Skills
* Aptitude
* Communication Skills

Output:

* Performance score out of 100

2. Logistic Regression:
   Predict placement eligibility:

* Score >= 75 → Eligible
* Score < 75 → Not Eligible

3. Support Vector Machine (SVM):
   Recommend best career domain:

* AI/ML
* Data Science
* Web Development
* Cybersecurity
* Cloud Computing
* UI/UX
* Software Development

4. K-Means Clustering:
   Group students into:

* Beginner
* Intermediate
* Advanced

Use clustering based on:

* Coding skills
* Aptitude
* Communication
* Performance score
